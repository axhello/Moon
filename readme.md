# Moon Themes

typecho主题Moon，极简主义风格主题，额~ 也说不上极简，但确实不繁杂！

## 主题设置

启用主题后，可前往设置外观->设置社交链接, 设置后会在主页显示。

## 更改css

😷sass文件有待优化~

css使用sass编译，如果你想更改css文件，自定义或者是什么的，可到sass更改自定义css，通过

```
npm install
```

安装完，后使用`gulp sass`编译。
